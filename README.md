# Description

This repository contains solutions and exercises developed as part of the 42 Piscine Reload program.

The main goal of this project is to reinforce and review core programming fundamentals introduced during the 42 Piscine, with a stronger focus on C programming, problem-solving, and algorithmic thinking.

The exercises cover a wide range of foundational topics such as:

- Basic C syntax and control structures
- Functions and modular programming
- Pointers and memory management
- String manipulation
- Arrays and data structures
- Algorithms and logical problem solving

Each exercise is designed to strengthen understanding of low-level programming concepts and prepare for more advanced projects within the 42 curriculum.

# Instructions

## Requirements

- GCC or any compatible C compiler is required
- Code must respect the 42 Norm (Norminette rules where applicable)
- Standard C library usage only (unless otherwise specified per exercise)

---

## Clone the Repository

```bash
git clone https://github.com/felipehillebrand-ops/Piscine-Reload.git
cd Piscine-Reload
```

---

## Compilation

Each exercise is placed in its own directory. To compile a specific exercise, use:

```bash
cc -Wall -Wextra -Werror exercise.c <other_files>.c
```

Example:

```bash
cc -Wall -Wextra -Werror ft_putchar.c
```

---

## Execution

After compiling, run the program:

```bash
./a.out
```

Each exercise may have its own expected input/output behavior, described in the subject.
